Eclipse p2 Update Site

MrMarsov MIK32 Eclipse Tools

Eclipse plug-in for MIK32 development. Use this update site URL in Eclipse to install the plug-in through Install New Software.

Update Site URL

https://timmarsov.github.io/MIK32EclipseTools/
  1. Open Eclipse.
  2. Go to Help → Install New Software...
  3. Click Add...
  4. Paste the update site URL.
  5. Select MrMarsov MIK32 Eclipse Tools.
  6. Click Next → Finish and restart Eclipse.
